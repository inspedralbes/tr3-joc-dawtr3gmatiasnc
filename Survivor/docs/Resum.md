# Resum. Projecte Survivor
**Curs:** 2DAM 2025-2026 | **Autor:** [Matias Negrón Carranza]
**Objectiu:** Joc de supervivència on els usuaris cooperen defensant un Trono d'onades d'enemics intel·ligents.
**Arquitectura:**
Sistema Client-Servidor. El Client (Unity/C#) gestiona gràfics, col·lisions i el cervell IA enemic (ML-Agents). El Servidor (Node.js/WebSockets) sincronitza l'estat multijugador. El client envia accions per JSON i el servidor en fa el *broadcast* a la sala en temps real.
